# pi-v-sistema-seguranca

Resumo
Este trabalho foi desenvolvido a partir da análise da necessidade de segurança no dia-a-dia de pequenos comércios onde é necessário controle do fluxo de clientes e manuseio dos produtos vendidos, bem como evitar perdas e furtos, e relacioná-lo ao estudo e aprendizado das matérias de controle e automação, processamento digital de sinal e engenharia de software na formação universitária do grupo.
