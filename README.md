# Projeto: PI-V Sistema de Segurança

## 📄 Resumo
Este trabalho foi desenvolvido a partir da análise da necessidade de segurança no dia-a-dia de pequenos comércios, onde é necessário controle do fluxo de clientes e manuseio dos produtos vendidos, bem como evitar perdas e furtos. O projeto também se relaciona ao estudo e aprendizado das matérias de **controle e automação**, **processamento digital de sinal** e **engenharia de software** na formação universitária do grupo.

## 📖 Descrição
Este projeto implementa um sistema de segurança em CLP (Ladder) e Arduino, com lógica de:
- Liga/Desliga do sistema (X0)
- Sensor de alarme (X2)
- Botão de inibição do alarme (X1)
- Temporizador para controle de inibição (T0)
- Saídas: Y0 (sistema ligado), Y1 (sirene), Y2 (alarme luminoso)
